Каталог

Режим работы основной рабочей области «Каталог» предоставляет доступ к списку товаров, позволяет просматривать различную информацию о товарах, а также добавлять товары в корзину.  Окно рабочей области разбито на несколько разделов. На рисунке каждый раздел подписан цифрой. Ниже описано что отображается в каждом из разделов.

 

  1. Иерархическая структура каталога (горячая клавиша Ctrl-Z). Выбор категорий и далее подкатегорий в структуре каталога приводит к отображению товаров в окне просмотра списка товаров (3).  Иерархическая структура каталога содержит элементы двух видов. Они отображаются соответственно темно желтым и светло желтыми цветами. Элементы выделяемые темно желтым цветом представляют собой категории товаров содержащие в себе подкатегории. Светло-жёлтым цветом отображаются подкатегории содержащие непосредственно сами товары. При выборе структуре каталога такого элемента (подкатегории) окно списка товаров (3) заполняется товарами данной подкатегории. Выбор элементов в структуре каталога может производиться щелчком левой кнопки мыши или с помощью клавиш "Вверх", "Вниз и Enter .
  2. Раздел поиска и отбора товаров. В этом разделе мы можем задавать различные условия отбора в списке товаров (3).

o    Поиск по артикулу (горячая клавиша Ctrl-F) позволяет производить поиск по каталожным, оригинальным и кросс номерам. Поиск выполняется по нажатию клавиши Enter или при установке галочки "По артикулу". Поиск будет произведен только при четырех и более введенных символах. Осуществляется поиск по вхождению введенных символов в любой участок существующего номера (т.е. при введенных символах "5555" запчасти с каталожным номером "155552", "005555" и "55554321" будут найдены). При поиске не учитываются разделительные символы как введенные в строке поиска, так и существующие в номерах. Разделительными символами считаются символы "-", ".", "/", " ". Т.е. результат поиска строки "123.4/56-7 8" будет аналогичен поиску по строке "12345678". Поиск осуществляется по всему каталогу не зависимо от текущего вида запчастей выбранного в структуре каталога. При удачном поиске можно перейти к списку товаров нажав горячую клавишу F3 или с помощью щелчка мышки.

o    Поиск по части наименования (горячая клавиша Ctrl-N) позволяет производить поиск товаров по части наименования. Поиск выполняется по нажатию клавиши Enter или при установке галочки "По имени". Поиск будет произведен только при двух и более введенных символах. Осуществляется поиск по вхождению введенных символов в любой участок существующего наименования (т.е. при строке поиска "sprinter" запчасть с именем "Поршень SPRINTER CM611-612 (88mm-STD)" будет найдена). При поиске не учитывается какие буквы вводятся  большие или маленькие, т.е. результат поиска "SPRINTER" и "sprinter" будет одинаковым. Поиск "По имени" осуществляется только при выборе подкатегории в структуре каталога светло-желтого цвета, а также при выборе одного или нескольких фильтров по событиям, таких как "Производитель", "Новые", "Акции" и "Приход". При удачном поиске можно перейти к списку товаров нажав горячую клавишу F3 или с помощью щелчка мышки.

o    Поиск по производителю позволяет отобрать товары по конкретному производителю. Выбор производителя осуществляется с помощью списка производителей. Поиск осуществляется по всему каталогу не зависимо от выбранной категории товаров в структуре каталога. Поиск по производителю можно комбинировать с поиском по имени и фильтрами по событиям "Новые", "Акции" и "Приход".

o    Поиск по событию позволяет отбирать товары по соответствующим событиям с товарами в компании. Поиск осуществляется по нажатию на соответствующую кнопки или с помощью установки соответствующего флажка. Нажатие на кнопку "Новые" отбирает новые позиции в ассортименте компании. Список товаров выводится в разрезе дат поступления товаров на склад. Нажатие на кнопку "Акции" отбирает акционные позиции на складе компании. Нажатие на кнопку "Приход" отбирает позиции поступившие на склад компании в последнее время. Список товаров выводится в разрезе дат поступления на склад

  1. Раздел списка товаров (горячая клавиша F3) представляет собой таблицу с информацией о товарах. Список отображается в зависимости от выбраной подкатегории товаров в структуре каталога (1) и/или в соответствии с выполненным поиском в разделе (2). В таблице выводится следующая информация:
    • Кат.номер - каталожный номер товара
    • Наименование - наименование товара
    • Производитель - производитель товара
    • Купить - щелчок правой кнопкой мышки в этом поле выводит окно ввода количество товара и отправляет товар в корзину
    • Остаток - остаток товара на основном складе компании
    • Остаток партнера - остаток товара на партнёрском складе компании
    • Ожидается - ожидается поступление товара на основной склад в ближайшее время
    • Заказано - количество товара уже заказанное Вами и находящееся в корзине
    • Цена - цена товара с учетом введенной наценки (наценка устанавливается в меню Сервис) в выбранной валюте
    • Цена закупочная - Ваша закупочная цена (видимость колонки регулируется соответствующим пунктом меню в меню Сервис)
    • Валюта - выбранная валюта в которой отображается цена
    • Фото - наличие фотографии или фотографий у товара. Если в колонке отображается надпись "Фото" - фотография есть. Для просмотра фотографии необходимо щелкнуть правой кнопкой мыши на нажать горячую клавишу F6. Фотография скачивается с сервера т.е. для просмотра фотографии необходимо подключение к сети Интернет.

Текущий выбранный товар в списке отображается строкой светло-желтого цвета. Выбор товаров осуществляется мышкой или с помощью клавиш "Вверх"/"Вниз". При выборе товара в списке товаров обновляется информация в разделе 4 (т.е. информация об аналогах и оригинальных номерах отображается для текущего товара). Товары которые привязаны к соответствующим событиям (Новые, Акция, Приходы) выделяются в списке соответствующим цветом (цветом кнопки отбора по соответствующему событию, например Новые запчасти отображаются в списке зеленым цветом). Отправка товара в корзину выполняется либо нажатием клавиши Enter, либо двойным щелчком мыши, либо одиночным щелчком в колонке "Купить".

  1. Раздел аналогов и оригинальных номеров товаров (горячая клавиша Alt-Z) представляет собой таблицы с информацией об аналогах выбранного товара в списке (3) и об оригинальных номерах товара у производителей автомобилей.
    • Список аналогов отображает информацию аналогичную списку товаров (3) и работа с ним производится аналогично. Единственно отличие двойной щелчок на строке аналога производит переход в список товаров и позиционирование на данном товаре с списке.
    • Оригинальные номера - представляют таблицу с вариантами номеров производителей автомобилей для данного товара. Представляет собой таблицу с оригинальным номером и производителем автомобиля у которого данный номер применяется для текущего товара.
