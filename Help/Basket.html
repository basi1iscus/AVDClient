Корзина

Режим работы основной рабочей области  "Корзина" предоставляет доступ к текущему заказу. В корзине отображаются запчасти отобранные для заказа. Рабочая область содержит панель инструментов и таблицу с отобранными запчастями.


  1. С помощью панели инструментов над корзиной можно выполнить следующие операции:
    • Добавить - переключает рабочую область в режим "Каталог" для поиска запчастей и добавления их в корзину
    • Очистить корзину - очищает текущий заказ, если он Вам не нужен. Восстановить очищенную корзину невозможно.
    • Отправить в обработку - отправляет текущий заказ на обработку менеджеру компании. Возможно только при наличии подключения к Интернет. После отправки корзина очищается. Следить за процессом обработки заказа можно в режиме "Заказы и отгрузки".
    • Загрузить из Excel - открывается окно для загрузки заказа подготовленного в другой программе и сохраненного в файле формата Excel. Подробнее смотрите здесь.
    • Выгрузить в Excel - экспортирует (выгружает) текущий заказ в формате Excel.
  2. Таблица текущего заказа отображает следующую информацию:
    • Кат.номер - каталожный номер запчасти
    • Наименование - наименование запчасти
    • Производитель - производитель запчасти
    • Количество - количество запчасти в корзине
    • Цена - цена товара
    • Сумма - общая сумма всего количества запчасти
    • Х - нажатие на значек "Х" позволяет удалить соответствующую запчасть из корзины
