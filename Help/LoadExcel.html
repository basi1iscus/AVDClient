Импорт из файла Excel

При работе с программой AVD Client Вы имеете возможность загружать и отправлять заказы, подготовив их в ваших учетных системах и сохранив в файл в формате Excel.

 Войти в режим загрузки из файла можно с помощью меню "Заказы/Импорт из файла" или в разделе Корзина нажав на ссылку "Загрузить из Excel". Сразу после этого программа попросит Вас указать файл из которого необходимо выполнить загрузку. Структура файла не имеет значения, но в нем должны присутствовать колонки с каталожным номером и количеством для заказа. После выбора файла откроется окно для загрузки.

 


Перед загрузкой необходимо указать колонку в которой содержится каталожный номер, колонку с количеством товара и номер первой строки с которой необходимо начинать загрузку запчастей. После этого можно нажать на ссылку "Найти товар". В случае успешного поиска товара по каталожному номеру правая часть окна (столбцы зеленого цвета) заполнятся найденным товаром. Каталожный номер найденного товара окрашивается в зеленый цвет. Каталожный номер товара, который система не смогла найти окрашивается красным цветом.  На картинке выше это товар в строке 2. Вам придется вручную выбрать данный товар с помощью каталога. Для этого щелкните два раза на строке с ненайденным товаром и Вы попадете в каталог где с помощью поиска находите нужный Вам товар и выбрав его переносите в окно импорта. После выбора всех товаров нажав ссылку "Загрузить" Вы переносите заказ в корзину. Затем Вы можете дополнить его в обычном режиме и отправить на обработку менеджеру компании.
